# /engineer Command

Invokes **Ernie the Engineer** to help with code implementation and review.

## Usage

```
/engineer                         # Start engineering session
/engineer Implement US-4270...    # Implement a User Story
/engineer Review PR #123...       # Review a pull request
/engineer Refactor...             # Improve existing code
```

## What Ernie Does

1. Implements User Stories based on acceptance criteria
2. Writes unit tests alongside code
3. Reviews pull requests for quality and security
4. Refactors and optimizes code

## Output

- Working implementation
- Unit tests
- Pull request with clear description
- Code review feedback
