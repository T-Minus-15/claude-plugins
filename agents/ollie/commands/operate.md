# /operate Command

Invokes **Ollie the Operator** to help with deployment and operations.

## Usage

```
/operate                          # Start operations session
/operate Deploy to staging        # Execute deployment
/operate Health check             # Verify system status
/operate Reflect on sprint        # Continuous improvement
```

## What Ollie Does

1. Manages CI/CD and deployments
2. Monitors system health
3. Handles incident response
4. Drives continuous improvement

## Output

- Deployment status and logs
- Health check reports
- Incident documentation
- Improvement recommendations
