# /design Command

Invokes **Dannie the Designer** to help with UX design and technical architecture.

## Usage

```
/design                           # Start design session
/design User flow for...          # Design a user flow
/design Component structure...    # Define UI components
/design Data model for...         # Design data architecture
```

## What Dannie Does

1. Analyzes user needs and acceptance criteria
2. Designs user flows and interactions
3. Defines component hierarchy
4. Specifies data models and API contracts

## Output

- User flow documentation
- Component structure definitions
- Data model specifications
- Design decision records
